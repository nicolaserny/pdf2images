# PDF2Images

Small project to preview PDF files as images.
You can test it [here](https://rad-klepon-373c44.netlify.app/).

## Requirements

-   Node 18
-   Yarn

## Development

First, install the dependencies:

```bash
yarn install
```

Then, run the development server:

```bash
yarn dev
```

Run unit tests:

```bash
yarn test
```

Run lint:

```bash
yarn lint
```
